# 🗺️ Aetox Website Navigation & Content Map

**"The Aetox Conversion Funnel: จากผู้เข้าชมสู่พาร์ทเนอร์ทางธุรกิจ"**

เอกสารฉบับนี้อธิบายโครงสร้างเนื้อหาของเว็บไซต์ Aetox (Landing Page) เพื่อเป็นแนวทางในการเขียน Copywriting และการออกแบบ UI ให้สอดคล้องกับกลยุทธ์การปิดการขาย

---

## 1. ลำดับเนื้อหาเชิงกลยุทธ์ (The Funnel Structure)

เว็บไซต์ถูกออกแบบมาตามลำดับจิตวิทยาการตัดสินใจ ดังนี้:

### Section 1: Hero (The Hook)
- **วัตถุประสงค์:** สร้างความตื่นตาตื่นใจและประกาศอำนาจ (Authority)
- **ข้อความหลัก:** "ความไร้ประสิทธิภาพคือสิ่งที่คุณเลือกหยุดได้"
- **CTA:** ปรึกษาเรา / ดูบริการ

### Section 2: Pain (The Awareness)
- **วัตถุประสงค์:** ชี้ให้เห็นรอยรั่วและมูลค่าความเสียหายหากนิ่งเฉย
- **หัวข้อสำคัญ:** Production Loss, Revenue Leak, Human Error

### Section 3: Comparison (The Logic)
- **วัตถุประสงค์:** เปรียบเทียบความแตกต่างระหว่าง "โค้ดทั่วไป" กับ "สถาปัตยกรรม Aetox"
- **จุดเน้น:** ความแม่นยำ 99.9%, ความปลอดภัย Zero-Model Training, ROI การันตี

### Section 4: Simulation (The Proof)
- **วัตถุประสงค์:** พิสูจน์ด้วยการทำงานจริง (Show, Don't just Tell)
- **องค์ประกอบ:** การจำลองการประมวลผลข้อมูล RAG ใน 3.2 วินาที

### Section 5: Services (The Solutions)
- **วัตถุประสงค์:** นำเสนอเครื่องมือที่จะเข้ามาแก้ปัญหา
- **หมวดหมู่:** AI Agent, Automation, Enterprise Web

### Section 6: ROI Calculator (The Decision)
- **วัตถุประสงค์:** ให้ลูกค้ามีส่วนร่วมและเห็นตัวเลขความคุ้มค่าจริง
- **การทำงาน:** กรอกจำนวนพนักงานและเงินเดือนเพื่อดูผลลัพธ์การประหยัดต้นทุน

### Section 7: Security (The Trust)
- **วัตถุประสงค์:** ปิดความกังวลสุดท้ายเรื่องข้อมูลรั่วไหล
- **จุดเน้น:** มาตรฐาน AES-256 และนโยบายความเป็นส่วนตัว

## 2. โครงสร้างเมนูและการนำทาง (Navigation Structure)

| เมนู | เป้าหมาย (Destination) | สิ่งที่ลูกค้าจะได้รับ |
| :--- | :--- | :--- |
| **หน้าแรก** | `/` | ภาพรวมทั้งหมดของ Funnel |
| **บริการ** | `/services` | ข้อมูลเจาะลึกรายบริการ (AI, Automation, Web) |
| **ผลงาน** | `/authority` | ความเชื่อมั่นจากโครงการจริงและรางวัลระดับชาติ |
| **เรียนรู้กับเรา** | `/academy` | บทความความรู้ด้าน AI สำหรับผู้บริหาร |
| **ติดต่อ** | `/contact` | ช่องทางปรึกษาสถาปนิกโดยตรง |

## 3. มาตรฐานการเขียนคำ (Copywriting Standard)

1. **Focus on Outcomes:** อย่าขายแค่ฟีเจอร์ ให้ขาย "ผลลัพธ์" (เช่น อย่าบอกว่า AI เร็ว ให้บอกว่า "ประหยัดเวลาทีมบริหาร 90%")
2. **Authoritative Tone:** ใช้ภาษาที่ดูเชี่ยวชาญและมั่นคง
3. **No Fluff:** เนื้อหาต้องกระชับ เข้าประเด็น ไม่เยิ่นเย้อ
