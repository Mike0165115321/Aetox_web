## 📋 1. Project Overview
Aetox.dev ถูกพัฒนาขึ้นเพื่อเป็นเครื่องมือเชิงกลยุทธ์ในการสื่อสารนวัตกรรมซอฟต์แวร์และการนำ AI มาใช้ในภาคธุรกิจ โดยเน้นการนำเสนอข้อมูลที่ซับซ้อนให้เข้าใจง่ายผ่านการจำลอง (Simulation) และการวิเคราะห์ผลตอบแทน (ROI Analysis)

### Core Philosophy:
*   **Precision Engineering:** การพัฒนาด้วยมาตรฐานความปลอดภัยและสถาปัตยกรรมที่รองรับการขยายตัว (Scalable Architecture)
*   **Business Transformation:** การออกแบบที่เน้นการแก้ปัญหาทางธุรกิจจริง (Pain Points) ด้วยเทคโนโลยีสมัยใหม่
*   **Data-Driven Decision Making:** การนำเสนอข้อมูลแบบโต้ตอบได้ (Interactive) เพื่อประกอบการตัดสินใจของผู้บริหาร

## 🛠 2. Technical Capabilities

### 2.1 AI & Retrieval-Augmented Generation (RAG)
*   **Architecture Simulation:** ส่วนจำลองการทำงานของระบบ AI ที่แสดงขั้นตอนการดึงข้อมูลจากฐานข้อมูลภายในองค์กรมาประมวลผลอย่างปลอดภัย
*   **Logic Visualization:** การแสดงกระบวนการทำงานของ Workflow Automation เพื่อให้เห็นภาพรวมของประสิทธิภาพที่เพิ่มขึ้น

### 2.2 ROI & Analysis Engine
*   **Automation Simulator:** เครื่องมือวิเคราะห์ผลตอบแทนจากการลงทุนในระบบอัตโนมัติ โดยเปรียบเทียบจากตัวแปรด้านแรงงานและเวลาในการทำงานจริง
*   **Dynamic Visuals:** การแสดงผลข้อมูลในรูปแบบแผนภูมิและแอนิเมชันที่ตอบสนองต่อข้อมูลนำเข้าแบบ Real-time

## 🏗 3. Technology Stack & Standards
เราเลือกใช้เครื่องมือที่เสถียรและเป็นมาตรฐานสูงสุดในปัจจุบัน:
*   **Next.js 15 (App Router):** สถาปัตยกรรมเว็บแอปพลิเคชันยุคใหม่เพื่อความเร็วและความปลอดภัย
*   **Tailwind CSS 4.0:** ระบบการจัดการสไตล์ที่ยืดหยุ่นและรองรับ Responsive Design ขั้นสูง
*   **Framer Motion:** การจัดการแอนิเมชันที่นุ่มนวล เพื่อสร้างประสบการณ์ผู้ใช้ที่ดี (UX)
*   **Internationalization (i18n):** รองรับระบบหลายภาษาอย่างสมบูรณ์ผ่าน JSON Dictionaries

## 📂 4. Project Structure
การจัดโครงสร้างโค้ดที่เน้นความชัดเจนและการบำรุงรักษา:
*   `app/`: การจัดการ Routing และ UI Layer ตามมาตรฐาน Next.js
*   `components/`: ส่วนประกอบหน้าเว็บที่ถูกแบ่งเป็นโมดูลย่อย (Modular Design) เพื่อการนำกลับมาใช้ใหม่
*   `data/content/`: ส่วนจัดเก็บเนื้อหา (Content Layer) แยกจากตัวโค้ดเพื่อความง่ายในการอัปเดตข้อมูล
