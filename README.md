## 💎 1. PRIME DIRECTIVE: "THE CLOSING ENGINE"
**"Everything built here must balance Technical Excellence with Business Conversion."**

Aetox.dev ไม่ใช่แค่เว็บไซต์ Portfolio หรือ Corporate Website ธรรมดา แต่คือ **"Closing Engine"** ที่ถูกออกแบบมาอย่างแยบยลเพื่อเปลี่ยนผู้เยี่ยมชม (โดยเฉพาะ C-Level Executives และ Decision Makers) ให้กลายเป็นพาร์ทเนอร์ทางธุรกิจ 

เราใช้สถาปัตยกรรม **7-Step Conversion Funnel**:
1. **Hook:** ดึงดูดด้วยผลลัพธ์และตัวเลข
2. **Pain:** ชี้ให้เห็นถึงต้นทุนที่แฝงอยู่ (Hidden Costs)
3. **Trust:** สร้างความน่าเชื่อถือด้วยผลงานระดับประเทศ
4. **Engage:** ให้ผู้ใช้มีส่วนร่วม (เช่น เครื่องคำนวณ ROI / RAG Simulator)
5. **Money:** แปลงความสูญเสียเป็นมูลค่าทางการเงิน
6. **Logic:** พิสูจน์ด้วยหลักการทางวิศวกรรมที่จับต้องได้
7. **Close:** ปิดการขายด้วย Call to Action ที่ชัดเจน

## 🛠️ 3. THE TECH STACK
เราเลือกใช้เครื่องมือระดับอุตสาหกรรม (Industry Standards) เพื่อให้ระบบมีเสถียรภาพ รวดเร็ว และรองรับการสเกล:

- **Core Framework:** Next.js 15 (App Router) — สำหรับ SSR, SEO ขั้นสูง และ Routing ที่มีประสิทธิภาพ
- **Language:** TypeScript (`.tsx`, `.ts`) — การันตีความปลอดภัยของโค้ด (Type-Safety) 100%
- **Styling:** Tailwind CSS — ออกแบบ UI แบบ Custom ด้วย Utility-first approach
- **Animations:** Framer Motion — สร้าง Micro-interactions และ Data-flow animations ระดับพรีเมียม
- **Icons:** Lucide React — ชุดไอคอนที่สอดคล้องกับธีมเทคโนโลยี
- **Dictionary/i18n:** ใช้โครงสร้าง Object-based Dictionary (`/data/content/`) เพื่อจัดการเนื้อหา (Copywriting) แยกออกจาก Logic ของ UI

## 🚀 6. LOCAL DEVELOPMENT & SETUP

### Prerequisites
- Node.js (v18.17.0 หรือใหม่กว่า)
- npm หรือ yarn

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mike0165115321/Aetox_web.git
   cd Aetox_web
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Environment Variables** (ถ้ามี)
   สร้างไฟล์ `.env.local` ที่ root directory:
   ```env
   # ตัวอย่าง
   NEXT_PUBLIC_API_URL=http://localhost:3000/api
   ```

4. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```
   เปิด [http://localhost:3000](http://localhost:3000) เพื่อดูผลลัพธ์บนบราวเซอร์

## 📏 7. DEVELOPER GUIDELINES (STRICT RULES)
สำหรับนักพัฒนาทุกคนที่ร่วมสร้างแพลตฟอร์มนี้ โปรดยึดถือกฎต่อไปนี้อย่างเคร่งครัด:

1. **Modular Architecture:** แยก UI, Logic, และ Data ออกจากกันเด็ดขาด ไฟล์ `tsx` หนึ่งไฟล์ **ห้ามมีความยาวเกิน 300 บรรทัด** หากเกินให้แตก Component ย่อย
2. **Absolute Centering:** ส่วน Navbar และเมนูหลักต้องใช้ `absolute left-1/2 -translate-x-1/2` เพื่อให้อยู่กึ่งกลางเสมอ
3. **Z-Index Management:** Navbar ต้องมี `z-[100]` เสมอ เพื่อป้องกัน Animation ทะลุขึ้นมา
4. **Tactile Feedback:** ทุกปุ่ม (Buttons) และการ์ดที่คลิกได้ (Interactive Cards) ต้องมี Transition `active:scale-95`
5. **No Unauthorized Dependencies:** ห้ามนำเข้า Library ภายนอกที่ไม่ได้รับอนุญาต (ให้ยึด Tailwind, Framer Motion, Lucide เป็นหลัก)
6. **Executive Thai Language:** ภาษาไทยที่ใช้ในเว็บไซต์ต้องเป็นภาษาระดับผู้บริหาร (C-Level) กระชับ ทรงพลัง ห้ามใช้ภาษาที่ดูเป็นการแปลตรงตัวจากหุ่นยนต์
