## 🛠️ 3. THE TECH STACK
เราเลือกใช้เครื่องมือระดับอุตสาหกรรม (Industry Standards) เพื่อให้ระบบมีเสถียรภาพ รวดเร็ว และรองรับการสเกล:

- **Core Framework:** Next.js 15 (App Router) — สำหรับ SSR, SEO ขั้นสูง และ Routing ที่มีประสิทธิภาพ
- **Language:** TypeScript (`.tsx`, `.ts`) — การันตีความปลอดภัยของโค้ด (Type-Safety) 100%
- **Styling:** Tailwind CSS — ออกแบบ UI แบบ Custom ด้วย Utility-first approach
- **Animations:** Framer Motion — สร้าง Micro-interactions และ Data-flow animations ระดับพรีเมียม
- **Icons:** Lucide React — ชุดไอคอนที่สอดคล้องกับธีมเทคโนโลยี
- **Dictionary/i18n:** ใช้โครงสร้าง Object-based Dictionary (`/data/content/`) เพื่อจัดการเนื้อหา (Copywriting) แยกออกจาก Logic ของ UI

สถาปัตยกรรมของโฟลเดอร์ถูกแยกส่วนอย่างชัดเจน:

```text
├── app/                        # Next.js App Router (UI & Routing Layer)
│   ├── (home)/                 # หน้า Landing Page หลัก
│   ├── authority/              # หน้าแสดงผลงานและระดับความน่าเชื่อถือ (Gold Medal)
│   ├── services/               # หน้าบริการหลัก (AI, Automation, Web Systems)

│   └── layout.tsx              # Root Layout
├── components/                 # Shared UI Components
│   ├── navbar/                 # Refactored: Logic, Desktop, Mobile
│   ├── home/                   # Home-specific sections (Simulation, Comparison)
│   ├── ui/                     # Basic UI blocks (NumberCounter, etc.)
│   └── Navbar.tsx              # Connector component
├── data/                       # The "Brain" (Data Layer)
│   ├── content/                # เนื้อหาทั้งหมดแยกตามหน้า (TH/EN)
│   │   ├── services/           # ข้อมูลเจาะลึกของแต่ละบริการ
│   │   ├── projects/           # ข้อมูลผลงานทั้งหมด
│   └── dictionaries/           # Global Text (Navigation, Footer)
└── public/                     # Static Assets (Images, Icons)
```

## 🚀 6. LOCAL DEVELOPMENT & SETUP

### Prerequisites
- Node.js (v18.17.0 หรือใหม่กว่า)
- npm หรือ yarn

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mike0165115321/Aetox_web.git
   cd Aetox_web
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Environment Variables** (ถ้ามี)
   สร้างไฟล์ `.env.local` ที่ root directory:
   ```env
   # ตัวอย่าง
   NEXT_PUBLIC_API_URL=http://localhost:3000/api
   ```

4. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```
   เปิด [http://localhost:3000](http://localhost:3000) เพื่อดูผลลัพธ์บนบราวเซอร์

## 📏 7. DEVELOPER GUIDELINES (AETOX STANDARDS)

1. **Executive Thai Language:** ภาษาต้อง "ทรงพลัง" และ "กระชับ" แบบผู้บริหาร (Executive Tone) ห้ามใช้ภาษาที่ดูเป็นการแปลตรงตัว
2. **Aetox Branding Rules:**
   * `shadow-aetox-glow`: ใช้กับปุ่ม CTA หลักเท่านั้น
   * `animate-aetox-breathe`: ใช้กับโลโก้ในหน้า Hero เพื่อสื่อถึงความ "มีชีวิต"
   * `text-aetox-accent`: ใช้เน้นข้อความไม่เกิน 1-2 คำต่อหน้า
3. **Modular Architecture:** แยก UI, Logic, และ Data ออกจากกันเด็ดขาด ไฟล์ `tsx` หนึ่งไฟล์ **ห้ามยาวเกิน 250 บรรทัด**
4. **No Placeholders:** ห้ามใช้ภาพ Placeholder ให้ใช้ `generate_image` เพื่อสร้าง Visual ที่ตรงกับแบรนด์เสมอ
5. **Tactile Feedback:** ทุกปุ่ม (Buttons) และการ์ดที่คลิกได้ ต้องมี Transition `active:scale-95` เพื่อความพรีเมียม
